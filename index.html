Hacking Docker Containers

Using Metasploit and NodeJS
This project demonstrates security implementations in a Docker container
through a NodeJS web app by analyzing vulnerabilties in the system.

Phases

widgets

The first phase involves creating a NodeJS web app which will work into a Docker container. This app will contain basic functionalities and will be used to exploit the Docker container containing the app.

bug_report

The second phase will consist of exploiting the Docker container through the web app and creating a report on the various weaknesses in the system.

call_missed_outgoing

For the final phase, the report from the second phase will be used to implement security measures in the system and repeat the same attacks to demonstrate the effectiveness of the defenses.

About Project

This web app is made as part of J Component project for the course Internet Security Management (CSE3502) from VIT, Vellore. This repository contains only the web app part of the project.
